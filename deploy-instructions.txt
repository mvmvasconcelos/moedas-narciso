Instruções para Deploy do Projeto

1. Para desenvolvimento local (localhost:3000):
   ```powershell
   .\deploy.bat
   ```
   Isso irá iniciar o ambiente de desenvolvimento usando o docker-compose.yml

2. Para fazer deploy em produção na Vercel:
   ```powershell
   .\deploy-prod.bat
   ```
   
   Na primeira vez que executar, você precisará:
   - Fazer login na Vercel (uma janela do navegador abrirá automaticamente)
   - Confirmar as configurações do projeto
   - Verificar se as variáveis de ambiente estão configuradas:
     * NEXT_PUBLIC_SUPABASE_URL
     * NEXT_PUBLIC_SUPABASE_ANON_KEY

3. O script deploy-prod.bat irá:
   - Limpar caches anteriores (.next e .vercel/output)
   - Construir a imagem Docker com Vercel CLI
   - Montar seu código no container
   - Executar o deploy em produção
   - Passar as variáveis de ambiente necessárias

4. Após o deploy:
   - Verifique a URL gerada
   - Teste o login e as funcionalidades
   - Verifique os logs no painel da Vercel se necessário
